🔗 실시간 동기화 설정

두 기기에서 같은 데이터를 실시간 공유하려면 Firebase 무료 계정이 필요해요.

1
console.firebase.google.com → 새 프로젝트 만들기
2
좌측 메뉴 → Realtime Database → 데이터베이스 만들기 → 테스트 모드
3
프로젝트 설정(⚙️) → 내 앱 → 웹앱(</>) 등록
4
표시된 firebaseConfig 값을 이 HTML 파일 상단에 붙여넣기
5
저장 후 두 기기에서 같은 파일 열기 → 실시간 동기화 완료 ✅
